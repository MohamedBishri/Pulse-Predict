# How PulsePredict Answers Hackathon Judging Criteria

## Innovation
- Explainable output, not a black-box score.
- Model-comparison laboratory.
- Bilingual interface.
- One architecture that can expand to multiple chronic diseases.

## Technical quality
- React + Vite frontend.
- FastAPI backend with generated OpenAPI documentation.
- JWT authentication and password hashing.
- SQLAlchemy persistence.
- Multiple ML models and evaluation metrics.
- Docker Compose for reproducible startup.

## User experience
- Premium responsive interface.
- Guided assessment.
- Demo data for rapid presentations.
- Risk visualization and factor ranking.
- History and printable report.

## Impact
- Helps users understand combined risk indicators.
- Supports early awareness and clinician conversations.
- Scales to wellness programs and preventive-health initiatives.

## Honest limitation
The packaged dataset is synthetic and intended for a functional prototype. Do not claim clinical validation. The winning narrative is that the product and ML pipeline are ready to replace the demonstration dataset with an approved real dataset and undergo external validation.
